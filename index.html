Learn More About FairPy

What is Python Library?

FairPy is a comprehensive Python Library for Machine Learning Fairness, covering various fairness notions, data structures, and learning tasks. FairPy is a good toolkit for mitigating bias and helping to deliver equitable outcomes from machine learning models.

FairPy includes many latest fairness algorithms, such as LabelBias[JN20] and FairGLM[DPM+22]. For consistency and accessibility, FairPy is developed on top of scikit-learn and PyTorch.

Sample Code of FairPy

Getting started with FairPy is simple. You can begin by installing the library, following the tutorial, and experimenting with the sample code provided in the documentation. This will help you understand the basic concepts and learn how to use FairPy effectively in your projects.

			
	from fairpy.dataset import Adult
	from fairpy.model import LabelBias

	# Choose or customize a dataset
	dataset = Adult()          
	# Extract features, labels, and sensitive attributes from the dataset        
	split_data = dataset.split()
	# Choose a fairness algorithm       
	model = LabelBias()               
	# Fit model with provided data 
	model.fit(split_data.X_train, split_data.y_train, split_data.s_train)
	# Predict  
	model.predict(split_data.X_test)
